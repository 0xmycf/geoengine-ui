Raster Scaling / Unscaling

This operator applies scaling or unscaling to a raster data set.
Scaling multiplies a raster data set with a scale value and adds a offset value: new = old * scale + offset . Unscaling is the inverse of the scaling operation: new = (old - offset) / scale . docs.geoengine.io.
